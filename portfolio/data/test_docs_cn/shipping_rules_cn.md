# 发货与物流规则

## 一、现货发货规则

- 现货商品在付款成功后，通常会在 `24小时内` 完成发货。
- 每日 `18:00` 之后确认的订单，通常会顺延至下一个工作日处理。

## 二、预售商品规则

- 预售商品通常会在商品正式开售后的 `48小时内` 发货。
- 若订单中同时包含现货商品和预售商品，系统可能会进行拆单发货。

## 三、延迟发货说明

- 如遇暴雨、台风、仓库搬迁或大促活动，发货时效可能延迟 `1-2天`。
- 若物流状态超过 `48小时` 未更新，客服应创建物流跟进工单并协助查询。

## 四、客服回复口径

- 当用户询问“现货多久发货”时，客服应说明：现货商品一般会在付款成功后 `24小时内` 发出。
- 当用户询问“预售商品多久发货”时，客服应说明：预售商品通常会在开售后 `48小时内` 发货。
- 当用户催单时，客服可结合订单时间判断是否属于正常待发货状态。
