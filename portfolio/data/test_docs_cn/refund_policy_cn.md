# 退款规则说明

## 一、退款到账时效

- 已付款但未发货的订单，审核通过后，退款通常会在 `1-3个工作日` 内原路退回。
- 如遇大促活动、节假日或银行处理延迟，退款到账时间可能延长至 `3-5个工作日`。
- 若用户使用信用卡支付，实际到账时间还可能受发卡银行处理速度影响。

## 二、退款适用范围

- 未发货订单支持直接申请取消并退款。
- 已发货订单一般需要用户签收后，再根据售后流程申请退货退款。
- 定制类商品、明确标注不支持无理由退货的商品，若无质量问题，则不支持无理由退款。

## 三、客服回复口径

- 当用户询问“退款多久到账”时，客服应说明：退款审核通过后，一般会在 `1-3个工作日` 内原路退回。
- 当用户在大促期间询问退款进度时，客服应说明：由于平台结算和银行处理时间影响，到账时间可能延长到 `3-5个工作日`。

## 四、补充说明

- 若用户重复提交退款申请，以系统最终审核通过的申请记录为准。
- 若系统显示退款失败，客服应协助用户核对收款账户信息和支付渠道信息。
